# Commit Message Template

Use this structure:

Pattern: Problem Name — Approach Used

Example:

Arrays: Two Sum — Hash Map Solution
Sliding Window: Longest Substring Without Repeating Characters — Optimized Window
Trees: Validate BST — Inorder Traversal

---

Optional Extended Version:

Pattern: Problem Name

- Added brute force solution
- Added optimal solution
- Documented complexity
- Updated pattern index
- Updated progress tracker
