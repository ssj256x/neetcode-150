# 🔥 How to Use This File

When stuck:

1. Identify what category the problem belongs to.
2. Review 3–5 solved problems from that pattern.
3. Extract the structural similarity.
4. Re-solve without looking at code.

Patterns repeat.
The surface story changes.
The structure does not.
